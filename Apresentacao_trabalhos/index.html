Portfólio da disciplina

Este site reúne os trabalhos desenvolvidos ao longo do semestre na disciplina de Design de Interação, utilizando HTML, CSS e JavaScript.

Sobre a disciplina

Design de interação aborda conceitos de HTML, CSS e JavaScript para criação de páginas e aplicações dinamicas.

Os trabalhos práticos desenvolvem habilidades progressivamente ao longo do semestre.

Estrutura do portfólio

O site está organizado em seções dedicadas a cada trabalho: cada um em sua própria página.

A navegação superior permite acesso rápido a qualquer seção do site.
