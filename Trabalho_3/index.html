Trabalho 3 – Consulta de API em Tempo Real

Como funciona?

Digite o nome de um Pokémon (Limpo ou um número de ID) no campo abaixo e clique em "Consultar".

O navegador enviará o dado informado para uma API externa (PokeAPI) usando o comando fetch. A API processará o pedido e responderá com os detalhes técnicos estruturados, que nossa página lê e traduz de forma simples para você abaixo.

Aguardando sua consulta... Os dados aparecerão aqui.
